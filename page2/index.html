Built an Image Classifier with FastAI and Deployed on HuggingFace

This is an end-to-end machine learning project for image classification, trained on the '5 Faces' dataset from Kaggle. The model, built with FastAI and fine-tuned using a ResNet-18 pretrained model, classifies images into one of five categories: Elon Musk, Jack Ma, Bill Gates, Narendra Modi, and Donald Trump. A Gradio-powered front-end lets users upload images and get instant predictions, seamlessly deployed on Hugging Face Spaces.
