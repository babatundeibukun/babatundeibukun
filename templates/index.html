Trend analysis of the US Presidential State Congress Speech (1790-2019) Using NLP

This project leveraged advanced Natural Language Processing (NLP) techniques to analyze over 200 years of congressional speeches, spanning from George Washington’s address in 1790 to Donald Trump’s in 2019. Every year, U.S. presidents outline the nation’s state and priorities in a pivotal address to Congress. These speeches offer a unique window into the shifting priorities and challenges of American leadership across eras.

Review-X: A Research Assistant Bot using LangGraph and OpenAI API

The 'Review-X' project addresses the challenge of controllability in generative AI systems by developing a research assistant bot leveraging LangChain and large language models (LLMs). This lightweight, multi-agent system automates research workflows through a user-defined process, creating specialized AI 'analysts' to investigate distinct subtopics. Key innovations include human-in-the-loop refinement for aligning AI outputs with user intent, simulated 'interviews' with LLMs using Retrieval Augmented Generation (RAG) and external sources, and parallel information synthesis using a map-reduce paradigm.
