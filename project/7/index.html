Project Overview:

This project presents a complete deep learning solution for classifying images into five categories: Elon Musk, Jack Ma, Bill Gates, Narendra Modi, and Donald Trump. Built with FastAI and leveraging transfer learning with a ResNet-18 model, the system achieves high accuracy while integrating a user-friendly web interface for real-time image predictions.

Key Highlights of the Project:

1. Transfer Learning with ResNet-18:

  • Utilized FastAI’s high-level API for rapid prototyping and training.
  • Applied data augmentation and presizing techniques for better generalization and improved accuracy.
  • Achieved 97% accuracy after fine-tuning for just 4 epochs.

2. Seamless Web Deployment:

  • Deployed the model using Gradio for an intuitive web interface.
  • Hosted the application on Hugging Face Spaces, enabling public access with no local setup required.

3. Real-Time Image Predictions:

  • Users can upload images directly through the web app and receive instant classification results.
  • Added example images from the dataset to let users test the app without uploading external images.

4. Robust Training Workflow:

  • Built a data pipeline with automatic train-validation splits.
  • Used FastAI’s training loop with layer freezing, unfreezing, and differential learning rates for optimal performance.

5. Overcoming Deployment Challenges:

  • Resolved authentication issues with Hugging Face Spaces by properly configuring access tokens.
  • Successfully pushed the model and app to production after iterative debugging and testing.

This project showcases my expertise in deep learning, model fine-tuning, web deployment, and end-to-end development. It’s a great example of how AI can classify images with impressive accuracy — even in a small dataset scenario.
